Using a dynamic route is really simple, you just have to replace the provider and model name while querying ai gateway to that of your smart route while prepending `dynamic-route`.

## Examples

### Fetch

```bash
curl https://gateway.ai.cloudflare.com/v1/{account_id}/{gateway_id} \
  --header 'Content-Type: application/json' \
  --data '[
  {
    "provider": "dynamic-route",
    "endpoint": "<name-of-your-route>",
    "headers": {
      "Authorization": "Bearer {cloudflare_token}",
      "Content-Type": "application/json"
    },
    "query": {
      "messages": [
        {
          "role": "system",
          "content": "You are a friendly assistant"
        },
        {
          "role": "user",
          "content": "What is Cloudflare?"
        }
      ]
    }
  },
```
